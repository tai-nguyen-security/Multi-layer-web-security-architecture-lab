# Multi-layer Web Security Architecture Lab

Role: Security Lab Engineer

This project demonstrates layered web security architecture using DVWA, ModSecurity (OWASP CRS), and CrowdSec.

## Key Activities

- Deployed DVWA vulnerable web application on Ubuntu VM
- Configured Nginx reverse proxy
- Integrated ModSecurity WAF protection
- Implemented CrowdSec automatic IP blocking
- Simulated SQL Injection and XSS attacks
- Monitored security logs and alerts

## Environment

Ubuntu Server
Apache
MariaDB
Nginx
ModSecurity (OWASP CRS)
CrowdSec
VMware
